The jquery-geocodable plugin provides a simple way to present a geocodable text input field.

US Only

$('#usonly').geocodable();

International

By default, the code limits results to US territories. By setting the 'countries' option to an empty array, this restriction is removed.

$('#international').geocodable({countries:[]});

International with Trigger


$('#withtrigger').geocodable({
  countries: [],
  choiceCallback: function(mark) {
    $('#result').text(mark.address);
  }
});
